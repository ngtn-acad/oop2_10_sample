# クエスト派遣管理アプリケーション
第10回 講義内資料サンプルコード

## 内容（メモ）
- データ(models)
  - キャラクタ
    - 名前（入力）
    - 性別（選択）
  - 武器
    - 名前（入力）
    - 攻撃力（入力）
    - 武器タイプ（選択）
  - クエスト
    - クエスト名（入力）
    - キャラクター（選択）
    - 武器（選択）
    - クエスト受注日（自動）

- 画面
  - キャラクタ一覧 
    - キャラクタ制作画面
    - キャラクタ編集
  - 武器リスト
    - 武器制作画面
    - 武器編集画面
  - クエスト画面
    - クエスト追加画面
    - クエスト編集画面
